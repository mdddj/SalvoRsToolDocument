# Salvo Api Window


## 1.窗口预览

1. 它会扫描项目中的router定义的api,并显示到这个窗口中

2. 点击列表项可以跳转到代码的位置

![salvo_api_window.jpg](salvo_api_window.jpg)


> 也可以在全局搜索中搜索api,就是右上角的那个搜索图标

## 2.右键菜单操作

右键单击某个项目,可以进行对应的操作

1. 复制 antd request
2. 复制 URL

> 后续会新增更多功能,有需求提交issue
